## How to use

1. Clone this repository:
    ```bash
    git clone https://github.com/sumanthd032/DSA.git
    cd DSA/respective-section/
    ```

2. Compile any `.cpp` file using:
    ```bash
    g++ filename.cpp -o filename
    ./filename
    ```
